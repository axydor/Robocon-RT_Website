Robocon-RT

Robocon-RT provides a real-time framework that facilitates the design and construction of remote controllable and autonomous mobile robot platforms.

The originality and innovation for this project comes from its modularity, robustness, execution efficiency and its reliance on more modern control protocols such as EtherCAT.

What We Do

Real-time robot control is a relatively hot topic in recent literature. However, there has not been any effort to create a modular, real-time robot control framework primarily focusing on EtherCAT technology

Currently available examples and tools generally require custom development of a large number of components to deploy a robot platform and are hence quite prohibitive due to the large initial investment required. The primary differences of this project from existing examples comes from its performance, modularity and the provision of existing components to facilitate robot design and deployment. Expected users of this product will be the programmers who are interested in robotics.

This project is supported by Teknolus, which is a company located in Ankara specialized in producing custom servo motors. Necessary hardware components, such as single board computers, are supplied by the firm throughout the project development.

Our Objective

Simple & efficient robot control framework for embedded systems

A robust network interface allowing remote access to robot platform

Support wide range of hardware components

Our Solution

Abstraction for writing user-defined modules
Managing modules with static-scheduling
Low-level motor control interface
Real-time communication with EtherCAT
A robust network interface
